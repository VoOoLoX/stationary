## Local development

- Install dependencies

```bash
bun i
```

- Set environment variables

In your `.env` file define following variables:
`SUPABASE_URL`
`SUPABASE_KEY`

```env
SUPABASE_URL='{YOUR SUPABASE URL}'
SUPABASE_KEY='{YOUR SUPABASE KEY}'
```

- Start development server

```bash
bun run dev
```
